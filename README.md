# Joomla-VM-ru-404-Forcer
Плагин для Joomla с компонентом Virtue Mart. Если у вас тоже не отдает 404 ошибку при изменении url, а вместо этого отдает предыдущую по сегментам url страницу, то вам нужен этот плагин. 
На данный момент плагин сделан только под ru версию, сайта. Использует слаги только _ru_ru таблиц, если у вас другой язык на сайте и другие таблицы - плагин работать не будет. 
Плагин работает сам по себе, достаточно загрузить его к себе на сайт и включить. 
В настройке плагина также есть строка для добавления кастомного файла error.php или любого другого файла вашей 404 ошибки.
